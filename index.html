Intro
Welcome to my website, where I will be posting all the worthy p5.js sketches I have produced. If you don't know what p5.js is, it's a javascript libary based on processing, a java based libary. It is formed on the idea of giving the user creative freedom, by making it easier for the user to interact with javascript, while still being able to use javascript's full potential. If you want to learn more, go to this site.

I program using p5.js as a hobby, as I find it intresting and fun. I am mostly using this site to archive and showcase my creations. Please mesasge me if you have any comments or questions.
